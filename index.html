So What Actually Happens?

  1. The brower goes to the DNS server and finds the real address of the server that the website lives on.

  2. The browser sends an HTTP request message to the server asking it to send a copy of the website to the client.

  3. if the server approves the client's request, the servder sends the client a "200 OK" message, which means "Permission granted! here you go!", and then starts sending the website's files to the browser as a series of small chunks.

  4. The browser assembles the small chunks into a complete website and displays it to you!
